# Clinical Documents for Ingestion

Coloque aqui os arquivos PDF ou de texto que contêm o conhecimento clínico (diretrizes, protocolos, artigos) que você deseja ingerir no cérebro vetorial da Cloudflare (Vectorize).

## Como usar:
1. Adicione os arquivos `.pdf` ou `.txt` nesta pasta.
2. Execute o script de ingestão apropriado (ex: `scripts/ingest-clinical-docs.mjs`) especificando o arquivo.
