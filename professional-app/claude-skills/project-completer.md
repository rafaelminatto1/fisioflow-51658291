# Skill: Project Completer

Especialista em levar projetos de software ao estado de "100% completo", focando em:
1. **Análise de Lacunas**: Identificação de TODOs e placeholders.
2. **Localização**: Garantir consistência idiomática (Português-BR).
3. **Automação de Fluxos**: Criar gatilhos inteligentes entre módulos (ex: Agendamento -> Financeiro).
4. **Estratégia de QA E2E**: Planejamento de testes Playwright para fluxos críticos.
5. **Documentação de Agente**: Manter pastas `.agent` atualizadas com o estado mental do projeto.
