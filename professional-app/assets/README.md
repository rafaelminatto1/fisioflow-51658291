# Assets

Substitua estes arquivos placeholder pelos seus icones reais:

## Icones Necessarios

| Arquivo | Tamanho | Descricao |
|---------|---------|-----------|
| `icon.png` | 1024x1024 | Icone principal do app |
| `splash-icon.png` | 512x512 | Icone da splash screen |
| `adaptive-icon.png` | 1024x1024 | Icone adaptativo (Android) |
| `favicon.png` | 48x48 | Favicon para web |
| `notification-icon.png` | 96x96 | Icone de notificacoes |

## Gerando Icones

Recomendamos usar o [Expo Icon Builder](https://buildicon.netlify.app/) ou criar no Figma.

Cores do FisioFlow Pro:
- Primary: #6366F1 (Indigo)
- Background: #FFFFFF
