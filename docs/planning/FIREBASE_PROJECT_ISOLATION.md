# Projeto FisioFlow - Isolamento do Firebase

Este projeto está configurado para usar o projeto Firebase: **fisioflow-migration**.

## Importante
Este projeto é SEPARADO do FireFlow. Não copie chaves de um para o outro.

## Variáveis de Ambiente
As chaves estão configuradas em .env e .env.local.

## Comandos Recomendados
Para garantir que o Firebase CLI está no projeto certo:
firebase use fisioflow-migration
