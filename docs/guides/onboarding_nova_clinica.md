# Onboarding de Nova Clínica - FisioFlow

Este guia contém o checklist passo a passo para configurar uma nova clínica no FisioFlow para garantir a extração do máximo de valor desde o dia 1.

## 1. Cadastro Inicial
- [ ] Criar a conta da organização no painel admin.
- [ ] Convidar o administrador da clínica via e-mail.
- [ ] Preencher dados básicos (Nome, Logo, Telefone, Endereço).

## 2. Configurações Financeiras e NFS-e
- [ ] Acessar `Faturamento > Parâmetros Fiscais`.
- [ ] Completar as 4 etapas do Wizard de NFS-e:
  1. Preencher CNPJ e Razão Social.
  2. Fazer upload do Certificado Digital A1.
  3. Emitir uma nota de homologação (R$ 1,00 sem validade fiscal).
  4. Ativar emissão em produção.

## 3. Agenda e Profissionais
- [ ] Cadastrar os fisioterapeutas.
- [ ] Definir a carga horária e horários de atendimento (blocos de 30/60 min) de cada profissional.
- [ ] Configurar pacotes de sessão comuns da clínica.

## 4. Automações de WhatsApp
- [ ] Solicitar ativação do número junto à API do WhatsApp Business.
- [ ] Configurar os templates de mensagens (`Lembrete D-2`, `Reengajamento`, `NPS Pós-Alta`).
- [ ] Ativar o robô de lembrete diário (disparo automático às 06:00 BRT).

## 5. Aplicativo Móvel (Paciente)
- [ ] Orientar a clínica a instruir seus pacientes a baixarem o "FisioFlow Paciente" nas lojas (App Store / Google Play).
- [ ] Fixar na recepção da clínica o Display com o QR Code para download do aplicativo.
